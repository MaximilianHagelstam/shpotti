# Shpotti

## Getting Started

```
git clone https://github.com/MaximilianHagelstam/shpotti.git
cd shpotti
yarn install
yarn prepare
yarn build
yarn start
```

## Stack

- Next.js
- TypeScript
- React Query
- Tailwind CSS
- Chakra UI
- Cypress
- Firebase
