# Shpotti

## Prerequisites

- Node.js 12<
- Yarn

## Getting Started

```
git clone https://github.com/MaximilianHagelstam/shpotti.git
cd shpotti

yarn install
yarn prepare
yarn generate:prod
yarn build
yarn start
```

## Stack

- Next.js
- Tailwind CSS
- MongoDB
- Prisma
