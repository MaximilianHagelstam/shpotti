# Shpotti

## Getting Started

### Prerequisites

- Node.js
- Yarn
- Docker

```
git clone https://github.com/MaximilianHagelstam/shpotti.git
cd shpotti
```

### Development

```
yarn install
yarn prepare
yarn dev
```

### Production

```
docker build . -t shpotti
docker run shpotti
```

## Stack

### Frontend
